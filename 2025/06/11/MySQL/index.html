MySQL

1. 基本概念

  • DMBS（Database Management System）：对数据库进行管理和操作

MySQL是一种 CS 架构的 DMBS。安装的MySQL包括客户端和服务端

C：mysql命令行，sqlyog（图形界面），执行增删改查操作（类比成 Linux 系统的终端）

S：安装Mysql的地方，对数据库进行处理

  • SQL（Structure Query Language）：结构化查询语言

注意在结尾加分号，才能执行

  • 主键：某一列，作用：唯一地标识某一行

  • USE xx（某数据库）

在sqlyog中点击数据库时，就会执行这个语句

  • SHOW

SHOW DATABASE：显示数据库

SHOW TABLES：显示表

DESCRIBE xx（表名）：查看表的数据属性

SHOW CREATE DATABASE xx：查看创建数据库的语句

SHOW CREATE TABLE xx：查看创建表的语句

HELP SHOW

  • DISTINCT：返回值不同的行

  • LIMIT：限制返回几行

  • OEDER BY ：对查询结果进行排序

DESC（ descending ） 降序排序，只作用于它前面的列

  • OFFSET：跳过结果集中的前几行
  • GROUP BY：对结果集中的行进行分组，和聚合函数一起使用

例如 COUNT(*)：表示计算每一组的行数，因为有 GROUP BY 所以才是计算每一组

2. 进阶概念

连接

INNER JOIN （ 内连接 ）：只返回两个表中匹配的记录

RIGHT JOIN（ 右连接 ）：返回右表中所有记录和左表中满足 ON 条件的记录，没有匹配的，则返回 NULL

FULL JOIN （ 全连接 ）：返回两个表中的所有记录，当一边没有匹配时，结果中的该边列返回 NULL

  • 使用子查询

子查询：嵌套在其他查询中的查询

多用于 WHERE 子句的 IN 操作符中

  • 联结表

外键：某个表中的一列，包含另一个表的主键值，定义了两个表的关系

联结关系是通过 WHERE 建立的，将两个表中相应匹配的行（ 通过外键 ）联结到一起
